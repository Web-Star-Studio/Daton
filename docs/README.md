# Base de Conhecimento do Daton

Este diretório concentra os artigos editoriais da knowledge base para cadastro manual em `/admin/base-conhecimento`.

## Como usar

1. Abra o arquivo do artigo.
2. Copie os campos do frontmatter:
   - `title`
   - `suggested_slug`
   - `category`
   - `summary`
3. Copie o corpo Markdown abaixo do frontmatter para o campo `bodyMarkdown` da UI.
4. Cadastre primeiro os artigos `P0`, depois `P1`, depois `P2`.
5. Publique e indexe artigo por artigo na UI.

## Critérios editoriais aplicados

- escopo global do produto, sem dados de organização específica
- linguagem orientada a fluxo e regras reais do sistema
- um tópico central por artigo
- headings e seções claras para favorecer recuperação por RAG
- limites atuais explicitados ao final
